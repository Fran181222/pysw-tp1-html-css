## 📖 Descripción del Proyecto

El sitio web simula la plataforma de una agencia turística donde los usuarios pueden explorar diferentes destinos, conocer las sucursales, leer artículos relacionados con viajes y ponerse en contacto con la agencia.

## Características Principales

Las vistas hechas:

- 🏠 **Home (`home.html`)**: Página de inicio con una vista general de los servicios de la agencia.
- 🗺️ **Destinos (`destinos.html`)**: Catálogo de los destinos turísticos ofrecidos.
- 🏢 **Agencias (`agencias.html`)**: Información sobre las distintas sucursales y oficinas físicas de la empresa.
- 📰 **Blog (`blog.html`)**: Sección de artículos y consejos de viaje.
- 💲 **Precios/Paquetes (`precios.html`)**: Listado de tarifas, promociones y opciones de paquetes turísticos.
- ✉️ **Contacto (`contacto.html`)**: Formulario de contacto para que los clientes puedan enviar sus consultas.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructurado con etiquetas semánticas (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
- **CSS3**: Estilos en cascada para diseño, colores, tipografía y animaciones.
- **Diseño responsivo**: Uso de Media Queries para adaptar el sitio a dispositivos móviles, tablets y computadoras de escritorio.

## ⚙️ Cómo visualizar el proyecto

Para ver y probar este sitio web de forma local. Solo sigue estos pasos:

1. **Clona este repositorio** en tu computadora:
   ```bash
   git clone https://github.com/Fran181222/pysw-tp1-html-css.git
   ```
2. **Navega al directorio** del proyecto:
   ```bash
   cd pysw-tp1-html-css
   ```
3. **Abre los archivos HTML**: Usa una extensión como *Live Server* en VS Code.
